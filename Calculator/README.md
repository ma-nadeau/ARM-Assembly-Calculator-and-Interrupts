# Calculator

## Overview
The calculator displays results on a seven-segment display.

## Input
- **On-Switches and Corresponding LEDs:**
  - Switches **0-4** represent the first number in binary.
  - Switches **5-9** represent the second number in binary.

## Push Buttons
- **0**: Reset
- **1**: Multiplication
- **2**: Subtraction
- **3**: Addition
